# Taxed HQ Guided Proficiency Path (v1)

## Program Structure
- **Format:** 3 modules, each with one practical build task and one validation checkpoint.
- **Completion model:** unlock next module only after passing module checkpoint.
- **Outcome:** user can independently model and explain their own tax scenarios.

## Module 1: Tax Foundations Through Your Own Paycheck

### Learning Objective
Understand how gross pay turns into take-home pay and identify your biggest tax levers.

### Lesson Components
- Quick concept brief: effective vs marginal rates.
- Guided baseline project build (`baseline_w2_year`).
- Interactive decomposition of paystub-like tax components.

### Project Task
Complete a baseline project and identify one change that could improve outcome.

### Completion Criteria
- Project saved with all required inputs.
- User correctly answers:
  - \"Does a higher bracket tax all your income?\" (must be \"No\").
  - \"What is your estimated effective rate?\"
- Confidence prompt >= 3/5.

---

## Module 2: Scenario Comparison and Decision Speed

### Learning Objective
Compare compensation scenarios quickly and explain why outcomes differ.

### Lesson Components
- Raise vs bonus walkthrough.
- Side-by-side comparison training with delta interpretation.
- \"Why changed\" cards mapped to each output shift.

### Project Task
Create and compare at least 2 scenarios in one project.

### Completion Criteria
- Baseline + at least one cloned scenario saved.
- User identifies preferred scenario and records reason.
- User correctly explains one tax delta in plain language.

---

## Module 3: Real-World Adjustment and Annual Strategy

### Learning Objective
Handle a mid-year tax change (job switch/bonus/side income) with confidence.

### Lesson Components
- Job-change withholdings reset playbook.
- Risk-band interpretation (`refund likely`, `balanced`, `owed likely`).
- Action checklist planning for next payroll cycles.

### Project Task
Complete `job_change_withholding_reset` scenario and finalize a 30-day action plan.

### Completion Criteria
- Project includes updated withholding assumption.
- User completes >= 3 action checklist items.
- Final confidence prompt >= 4/5.

---

## Scoring and Progression
- **Module score (100 max):**
  - 40 pts: task completion
  - 30 pts: concept validation answers
  - 20 pts: scenario explanation quality
  - 10 pts: action checklist completion
- **Path completion threshold:** 75+ score in each module.

## Certificates and Outcomes
- Badge awarded per module:
  - `Paycheck Decoder`
  - `Scenario Strategist`
  - `Tax Workflow Builder`
- End-state summary generated for user:
  - Completed projects
  - Scenario count
  - Top 3 learned tax levers

## Operational Notes
- Keep each module <= 15 minutes active use.
- Use plain language and avoid jargon density.
- Trigger reminders if a user pauses > 7 days between modules.
